ADA for Warriors

ADA for Warriors (ticker 4WARD) is a mission-driven single pool for Cardano. The pool was created as a way to give back to those who have quietly served by donating fifty percent of the pool rewards to the Third Option Foundation. All donations can be tracked on our main Swift Crypto LLC website.

The pool consists of one block producer with two relay nodes deployed to AWS. Each AWS instance is using 4 CPUs and 16 GB RAM, which exceeds the current hardware requirements. One relay and the block producer are in the AWS US East Region and the other relay is in the AWS European Central Region.

Security and recovery best practices were put in place to help guarantee server availability. Grafana is used to remotely monitor the status of all three servers. Should disaster recovery be needed, secure remote access is available to our admins from anywhere in the world.
