Steve Fisher, our pool operator, is a core member of Freeloaderz and the lead web developer for the FreeLoaderz SmartClaimz project.

FreeLoaderz was created in February of 2022. It brought together Cardano stake pools operators that were offering Cardano TX Submit API access to their Relay servers and set them up behind Load Balancers (LBs). The API service ultimately helps improve transaction speed by allowing supporting user wallets to submit transactions directly to a Relay through the LBs. As of July 2022, there are nearly 900 Cardano enthusiasts in the Discord server!

FreeLoaderz SmartClaimz is a novel reward distribution platform that aims to be the premier web3 dApp for end users to be able to claim tokens, and for decentralized autonomous organizations (DAOs), token projects, NFT projects, video game projects, and more to distribute quality tokens to users.
